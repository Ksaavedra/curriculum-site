# curriculum-site

Projeto fullstack de currículo e portfólio desenvolvido com Angular e Node.js.

# 📁 Estrutura

```txt
frontend → Angular
backend  → API Node.js
```

# 🚀 Tecnologias

## Frontend

- Angular
- TypeScript
- SCSS
- Angular Material
- RxJS
- Signals

## Backend

- Node.js
- Express
- Nodemailer
- Resend
- REST API

# 📌 Observações

Projeto em evolução contínua com melhorias de acessibilidade, responsividade, testes unitários e modernização visual.
