# Frontend - curriculum-site

Currículo e portfólio frontend desenvolvido com Angular e TypeScript.

# 🚀 Tecnologias

- Angular
- TypeScript
- HTML5
- SCSS
- Angular Material
- RxJS
- Signals
- Responsive Design
- REST API
- Dark Mode
- UX/UI
- Acessibilidade
- Jest
- Testes Unitários

# ▶️ Executar projeto

```bash
npm install
ng serve
```

Abrir no navegador:

```txt
http://localhost:4200
```
